Tipos de eventos

  • click: Se dispara cuando el usuario hace clic en un elemento (botón, enlace, div, etc.).
  • dblclick: Ocurre cuando el usuario hace doble clic sobre un elemento.
  • mouseover: Se activa cuando el puntero entra en el área de un elemento.
  • mouseout: Se dispara cuando el puntero sale del área del elemento.
  • mousemove: Ocurre cada vez que el ratón se mueve dentro del elemento.
  • mousedown: Se ejecuta cuando el usuario presiona un botón del ratón sin soltarlo.
  • mouseup: Se dispara cuando el usuario suelta el botón del ratón.
  • keydown: Se produce al presionar una tecla (antes de soltarla).
  • keyup: Se activa al soltar una tecla previamente presionada.
  • keypress: Se lanza cuando el usuario mantiene pulsada una tecla el tiempo suficiente para generar un carácter.
  • submit: Evento de formularios que ocurre justo antes de enviarse los datos.
  • change: Se dispara cuando el valor de un input cambia y luego pierde el foco.
  • focus: Ocurre cuando un elemento obtiene el foco (cursor dentro, por ejemplo).
  • blur: Se activa cuando un elemento pierde el foco.
  • load: Se ejecuta cuando un recurso (página, imagen, script…) termina de cargarse.
